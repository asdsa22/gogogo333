펫 시스템 무기를 환생시켜 펫으로 만들고 함께 전투하세요
🥚 펫 얻는 방법 (환생)
  1. 무기를 +20강까지 강화합니다
  2. !내정보 하단에 환생 버튼이 나타납니다
  3. 버튼을 누르면 무기가 펫으로 변환되고 강화 단계는 초기화됩니다
  4. 1환부터는 펫 레벨 140 (최대 레벨) 도달 후 다시 환생 가능
🔄
환생 시 초기화되는 것:
강화 단계, 강화 포인트, 무기 이름, 모든 전투 스탯, 스탯 포인트(10% 환급), 미수령 사냥 상자

환생 후에도 유지되는 것: 채팅 레벨, 보스 소환권, 환수(환생 횟수), 인벤토리 아이템 전체
📊 스탯별 전투 역할
❤️
체력 (내구력)
체력이 높을수록 크리티컬 발동 확률이 올라갑니다.
체력이 높은 펫은 한 번 터지면 큰 피해를 줍니다.
🛡️
방어력
방어력이 높을수록 크리티컬 데미지 배율이 올라갑니다.
크리가 터졌을 때의 한 방이 강해집니다.
⚔️
공격력
공격력이 높을수록 기본 데미지가 직접 올라갑니다.
가장 직접적으로 딜에 기여하는 스탯입니다.
⚡
순발력
순발력이 높을수록 공격 적중률이 올라갑니다.
빗나가면 데미지가 0이므로, 꾸준한 딜을 원한다면 중요합니다.
💡
스탯이 S급 기준치를 초과한 만큼 레이드 데미지에 추가 보너스가 붙습니다. 어떤 스탯이든 높을수록 유리합니다.
🔮 숨겨진 베이스 등급

펫의 각 스탯(내구력·공격력·방어력·순발력)에는 눈에 보이지 않는 베이스 등급이 있어, 레벨업 시 성장량에 영향을 줍니다.

F → D → D+ → C → C+ → B → B+ → A → A+ → S
  • 베이스 등급은 유저에게 비공개입니다 (총 10단계)
  • S등급에 가까울수록 레벨업 시 스탯이 더 많이 오릅니다
  • S급 베이스는 환생 후에도 유지됩니다 (1환 이상부터)
🔄 환수 시스템 (최대 5환)

환생할수록 펫의 성장률이 누적으로 증가합니다. 같은 레벨이라도 고환수 펫이 훨씬 강합니다.

나머지 베이스 등급은 재랜덤으로 새로 뽑힙니다.

✨ S+ 능력치 보정 (데미지 보너스)

펫 스탯이 S급 동레벨 기준치를 초과하면 초과분(능)만큼 레이드 데미지에 추가 보너스 %가 붙습니다.

계산 공식
❤️
내구력 초과
초과분 ÷ 4 (소수점 버림) = 보너스 %
⚔️
공격력 초과
초과 1당 +1% 보너스
🛡️
방어력 초과
초과 1당 +1% 보너스
⚡
순발력 초과
초과 1당 +1% 보너스
예시
체력 S기준 대비 +10 초과 → 10÷4 = 2%
공격력 +2 초과 → 2%
방어력 -5 미달 → 0% (음수 무시)
순발력 +3 초과 → 3%
합계: +7% 데미지 보너스
📊
!펫정보에서 스탯 옆에 표시되는 S+ 등급이 바로 기준치 초과 상태입니다.
음수 초과(기준치 미달)는 보너스에 영향 없습니다. S기준치는 환수 보정이 적용된 값을 기준으로 계산됩니다.

판매 / 아이템 교환 필요 없는 아이템을 포인트로 바꾸거나 다른 유저와 교환하세요
💰 아이템 판매
  • 명령어: !판매
  • 인벤토리에서 아이템을 선택하고 수량을 입력
  • 아이템 1개 = 강화 포인트 10P 전환
🤝 아이템 교환
  • 명령어: !교환
  • 서버 멤버 선택 → 내 아이템·수량 입력
  • 상대방 DM으로 교환 제안 전송
  • 상대방 수락 시 1분 내 최종 승인
  • 최대 1회 25개까지 교환 가능
⚠️
동시에 진행 중인 교환은 1건만 가능합니다. 교환 신청 후 아이템을 판매하거나 사용하면 교환이 자동 취소됩니다.
