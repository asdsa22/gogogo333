무기 강화
포인트로 무기를 +20강까지 강화하기
📌 기본 흐름
  • 채팅 → 경험치 쌓여서 레벨업
  • 레벨업 시 강화 포인트 +100P 지급
  • !강화 로 5P 소모해 강화 시도
  • +20강까지 가능 (실패해도 단계 안 내려감)
🎲 강화 특징
  • +10강부터 무기 이름 변경 가능
  • ⚡ 자동강화: 포인트 2배 소모 (10P/회)
  • +15강 이상 성공 시 스탯 포인트 추가 획득
  • 레이드 보상으로 강화권(배율) 획득 가능
🎁 무기 이름 변경
  • 조건: +10강 이상
  • 비용: 강화 포인트 10P
  • 명령어: !무기이름 <새이름>
  • 최대 15글자 입력 가능

보스 레이드
협력해서 보스를 무찌르고 보상 획득하기
📌 기본 흐름
  1. 매일 밤 12시에 소환권 10점 지급
  2. !보스소환 으로 원하는 등급 선택 후 소환
  3. 채널에 ⚔️ 공격! 버튼으로 계속 공격
  4. 시간 제한 안에 처치하면 전원 보상 획득
  5. 딜 1등 또는 참여율 1등은 아이템 2개 획득
🐉 보스 레이드 상세 정보 확인하러 가기 →
⚡ 패턴 시스템
  • 계산 문제 / 정답 입력 시 딜 2배 1분
  • 이모지 선택 / 정확한 이모지 클릭
  • 타자검정 / 문구 정확히 입력 (칠흑/지옥)
  • 색깔맞추기 / 지정 유저 전용 (4연속 또는 6연속)
🌟 지옥2 특징
  • 2개 페이즈로 구성
  • 1단계: 일반 패턴 없음 → 순수 공격
  • 1단계 클리어 후 → 2단계 시작
  • 2단계: 색깔 6연속만 나타남
  • 2페이즈 모두 클리어 시 초 고급 보상!
⚠️
주의: 7일 이상 보스를 소환하지 않으면 소환권이 0으로 초기화됩니다!

펫 시스템
무기를 환생시켜 강력한 펫으로 키우기
🥚 펫 획득 방법 (환생)
  1. 무기를 +20강까지 강화
  2. !내정보 에서 환생 버튼 클릭
  3. 무기가 펫으로 변환되고 레벨 1 초기화
  4. 1환부터: 펫 Lv.140 도달 후 다시 환생 가능
🔄 환수 보정 (최대 5환)
  • 환생할수록 성장률 증가
  • 0환: 기본
  • 1환: +10%
  • 2환: +20%
  • 3환: +30%
  • 4환: +40%
  • 5환: +50% 👑
🌟 펫 베이스 유지 시스템
  • 조건: 체·공·방·순 각 스탯이 S등급 베이스인 스탯
  • 판별: 레벨 140 기준 스탯이 높을수록 S베이스 확률 ↑
  • 보정: 환생 시 S베이스 스탯은 다음 환생에도 S베이스 유지
  • 팁: 환생마다 S베이스를 하나씩 맞추면 5환 때 올 S펫 확정 획득!
⚡ 무기추출 시스템
  • 조건: 5환 Lv.140 + +20강
  • 효과: 펫은 보관, 새 0환 펫 시작
  • 보상: 알 아이템 획득
  • 최대 20마리 보관 가능
📦 펫 보관
  • !펫정보 에서 펫 교체
  • 보관 중인 펫은 성장 멈춤
  • 사냥 / 레이드 중엔 교체 불가
  • 5환 미보관 펫 있으면 무기추출 불가
✨ 초기화 구슬 종류
  • 🔮 초기화구슬: Lv.1 초기화 + 베이스 재랜덤
  • 💫 완전초기화구슬: 새 펫으로 변경
  • 🎯 지정초기화구슬: 원하는 펫 직접 선택
  • 모두 환수 보정은 유지
